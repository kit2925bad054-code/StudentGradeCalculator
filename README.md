# Student Grade Calculator

## Description
Student Grade Calculator is a Java program that calculates a student's grade based on marks entered by the user. The program computes the total marks, average percentage, and assigns a grade according to the performance.

## Features
- Accepts marks for multiple subjects
- Calculates total marks obtained
- Computes average percentage
- Assigns grades based on percentage
- Displays results in a user-friendly format

## Technologies Used
- Java
- Object-Oriented Programming Concepts
